# Introduction to React

React is a JavaScript library for building user interfaces with reusable components.

## What You'll Learn

- Understanding components
- JSX syntax
- Props and State
- Hooks and effects

## Why React?

React makes it easier to create interactive UIs by breaking down complex interfaces into smaller, reusable pieces called components.

```javascript
// Your first component
function App() {
  return <h1>Hello, World!</h1>
}
```
